Algorithm List

(1) Fizz Buzz

This algorithm is a test of your use of conditional statement in combination with loops

Example

Write a function that takes a num as a value, the function logs out every number from one(1) to num. if the number to be logged is divisible by 3 log fizz instead, buzz if divisible by 5 and fizzBuzz if divisible by both 3 and 5

(2) Hameless Ransom Note

This algorithm is a test of your understanding of conditional statement, loops, runtime optimization/ time complexity (Big O notation)

Example

write a function that takes two parameters. The first parameter is a note we want to write out from a magazine text as a string, the second parameter is the magazine text as a string. The purpose is to know if there are enough words in the magazine text to form the note. If there are enough words, return true else false

(3) isPalindrome

This function takes in a string as an argument and return true if the can be spelt same in reverse order. Using array and string manipulation

(4) Ceasar Cipher

This function takes in 2 parameters, a string and number, the objective is to shift every letter of the string by the number that is passed in. the function also handles negative numbers to shift the letter backward

(5) Reverse Words

This takes in a string and reverse every word in the string and return the new string.

(6) Reverse Array

This takes in a array as a parameter and reverse the array and return the reversed array. -manipulate the array passed in. Don't push elements into a new array and retrun that array. Don't use the reverse array method also.

(7) Mean Median and Mode

A function takes in an array and returns and object containing the mean, median and mode of the array parsed

(8) Two Sum

A function takes in an array and a sum and returns an array of numbers of the pairs that add up to the sum. the result should be returned in an array of array
